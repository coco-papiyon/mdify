# mdify

クリップボード内のタブ区切りのデータをMarkdown表に変換するシンプルなツールです。

## 機能

- クリップボードからタブ区切りのデータを読み取り、Markdown形式の表に変換
- 表以外のテキストはそのまま保持
- コマンドラインオプションで柔軟な入出力対応
- デバッグモードで処理内容をログ出力

## インストール

### Windows

1. リポジトリをクローンまたはダウンロード
2. `install.bat` を実行して右クリックメニューに登録

### 手動ビルド

Go 1.22以上が必要です。

```bash
git clone https://github.com/coco-papiyon/mdify.git
cd mdify
go build .
```

## 使用方法

### 基本的な使い方

1. Excelやスプレッドシートからデータをコピー（タブ区切り）
2. コマンドを実行:
   ```bash
   ./mdify
   ```
3. Markdown表がクリップボードにコピーされます

### コマンドラインオプション

- `-input <ファイル>`: 入力ファイルを指定（省略時はクリップボードを使用）
- `-output <ファイル>`: 出力ファイルを指定（省略時はクリップボードに出力）
- `-dry-run`: クリップボードを変更せず、結果を標準出力に表示
- `-debug`: デバッグ情報を `mdify.log` に出力

### 例

#### 入力データ（クリップボード）:
```
製品名	価格	在庫
ノートパソコン	150000	10
マウス	3000	50
キーボード	5000	30
```

#### 実行:
```bash
./mdify -dry-run
```

#### 出力:
```
| 製品名 | 価格 | 在庫 |
|--------|------|------|
| ノートパソコン | 150000 | 10 |
| マウス | 3000 | 50 |
| キーボード | 5000 | 30 |
```

### 表とテキストの混在

表以外のテキストも含むデータを処理できます：

#### 入力:
```
会議資料

部署	人数	予算
営業部	20	5000000
技術部	15	8000000

備考: 予算は年間予定
```

#### 出力:
```
会議資料

| 部署 | 人数 | 予算 |
|------|------|--------|
| 営業部 | 20 | 5000000 |
| 技術部 | 15 | 8000000 |

備考: 予算は年間予定
```
